# Deutsch.app — Немецкий язык с AI

Персональное приложение для изучения немецкого с AI-диалогами, тестом уровня и профессиональными терминами.

## Структура проекта

```
deutsch-project/
├── index.html          # Главный файл — вся HTML-разметка
├── css/
│   └── style.css       # Все стили (переменные, компоненты, адаптив)
├── js/
│   ├── data.js         # Данные: уроки, словарь, сценарии, специальности
│   ├── state.js        # Состояние приложения (localStorage)
│   ├── onboarding.js   # Онбординг + тест уровня
│   ├── app.js          # Основная логика: dashboard, план, задания, словарь
│   ├── practice.js     # AI-диалоговый тренажёр (Claude API)
│   └── boot.js         # Инициализация приложения
└── README.md
```

## Запуск

Просто открой `index.html` в браузере. Никакого сервера не нужно.

> ⚠️ Для AI-практики нужен интернет (Claude API вызывается напрямую из браузера).

## Как добавить новый урок

В `js/data.js` в массив `GENERAL_LESSONS` добавь объект:

```js
{
  id: 'g7',
  title: 'Название урока',
  de: 'Немецкое название',
  color: 'c-gold',  // c-gold, c-green, c-blue, c-purple, c-teal, c-orange
  type: 'A1',
  words: [
    { de: 'das Wort', ru: 'слово', ex: 'Das ist ein Wort.', cat: 'general' }
  ],
  quiz: [
    { q: 'das Wort', opts: ['слово', 'буква', 'фраза', 'текст'], a: 0 }
  ]
}
```

## Как добавить новый AI-сценарий

В `js/practice.js` в массив `SCENARIOS` добавь:

```js
{
  id: 's_new',
  spec: 'it',        // 'general' | 'it' | 'med' | 'fin' | 'biz' | 'law' | 'eng'
  icon: '🖥️',
  title: 'Название',
  diff: 'med',       // 'easy' | 'med' | 'hard'
  desc: 'Описание сценария',
  tip: 'Совет: используй...',
  system: `Системный промпт для Claude. Описывает роль, правила, формат ошибок.`
}
```
