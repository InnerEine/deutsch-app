# DEUTSCH.app — Полная документация проекта

## 1. ОБЗОР ПРОЕКТА

**Название:** DEUTSCH.app  
**Репозиторий:** innereine.github.io/deutsch-app  
**Технологии:** HTML + CSS + JS (один файл index.html), Firebase, Gemini API  
**Цель:** Веб-приложение для изучения немецкого языка по методике CEFR (A1-B2)

## 6. МЕТОДИКА ОБУЧЕНИЯ

### 6.1 Научные принципы

```
1. Input первичен (70% времени — чтение и слушание)
   Теория Крашена: язык усваивается через понятный input
   
2. Output закрепляет (30% времени — письмо)
   Написание дневника каждый день — ключевая практика
   
3. Грамматика через паттерны
   Сначала видишь примеры → потом формулируешь правило
   НЕ наоборот (не правило → примеры)
   
4. Spaced Repetition для словаря
   Алгоритм Leitner: знаю → реже, не знаю → чаще
   
5. Interleaved Practice
   Микс грамматики, лексики и чтения в одном уроке
```

### 6.2 Структура уровней CEFR

```
A1 — Первые шаги     50 уроков  ~80 часов   2 месяца
A2 — Я справляюсь   50 уроков  ~100 часов  2 месяца
B1 — Я понимаю      50 уроков  ~120 часов  3 месяца
B2 — Я свободен     50 уроков  ~160 часов  4 месяца
─────────────────────────────────────────────────────
Итого:              200 уроков  ~460 часов  ~11 месяцев
```

### 6.3 Блоки по уровням

```
A1: Кто я? / Моя семья / Город / Еда и кафе / Время
    Грамматика: sein·haben·Akkusativ·Nominativ·Präteritum

A2: Работа / Квартира / Здоровье / Путешествия / Чувства
    Грамматика: Perfekt·Dativ·Modalverben·Komparativ·Konjunktionen

B1: Медиа / Карьера / Экология / История / Общество
    Грамматика: Konjunktiv II·Passiv·Infinitiv·Plusquamperfekt·Konjunktiv I

B2: Политика / Экономика / Наука / Искусство / Философия
    Грамматика: Nominalstil·Partizipien·Konzessiv·Modalpartikeln·Komplexe Sätze
```

### 6.4 Структура урока (5 шагов)

```
Шаг 1 — ОБЪЯСНЕНИЕ (3 мин)
  Краткое объяснение грамматики через примеры
  НЕ правило → примеры, а примеры → правило
  
Шаг 2 — ТЕКСТ/ДИАЛОГ (4 мин)
  Живой диалог по теме блока
  Все слова кликабельны (перевод RU+EN)
  Озвучка через Web Speech API
  Ключевая грамматика выделена золотым
  
Шаг 3 — УПРАЖНЕНИЯ (5 мин)
  5 упражнений СТРОГО по материалу урока
  Типы: truefalse / choice / input / order / translate
  После каждого — объяснение
  
Шаг 4 — СЛОВА (2 мин)
  8-10 новых слов из урока
  Перевод RU + EN для каждого
  Кнопка «+ В словарь»
  
Шаг 5 — НАПИСАТЬ САМОМУ (5 мин)
  3-5 предложений используя грамматику урока
  AI проверка через Gemini
  Подсветка ошибок красным
```

### 6.5 Экзамен уровня (стандарт Goethe Institut)

```
Часть 1: Чтение (Lesen)        25%
  Текст 150-300 слов + 5 вопросов на понимание

Часть 2: Аудирование (Hören)   25%  
  Текст озвучивается → 5 вопросов по услышанному
  Можно прослушать дважды

Часть 3: Грамматика             25%
  10 вопросов — только ввод текстом (не выбор)
  По грамматике всех 5 блоков уровня

Часть 4: Письмо (Schreiben)    25%
  A1: 30-50 слов / A2: 50-80 / B1: 100-150 / B2: 150-200
  AI проверка через Gemini

Порог прохождения: 60% в каждой части
Без ограничения времени
Пересдача: сразу, без ожидания
```
